# premix

## Installation:

Ubuntu (tested with Pop_OS! 20)
- `conda install -c conda-forge sox`
- `conda env create -f env.yaml` 

Windows
- download & install: https://sourceforge.net/projects/sox/files/sox/
- `conda env create -f env.yaml`
